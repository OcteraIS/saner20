# Saner 2020
## Replication package for `Cross-Dataset Design Discussion Mining`

### Environment
- `ubuntu==16.04`
- `python==3.6.9`
- `pip==18.1`

### Getting Started
- Please go to the root directory of the project
- Run `pip install -r requirements.txt`
- `python <filename.py>`
